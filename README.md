## üö¶ Getting Started

### Prerequisites

* Node.js (v18+)
* A Web3 Wallet (e.g., MetaMask)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/GitOnBoard.git

```


2. Install dependencies:
```bash
npm install

```


3. Run the development server:
```bash
npm run dev

```
